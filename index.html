What this site is about

This website is a small showcase of the projects and ideas I am working on while learning programming, web design, and computer science basics. It is not meant to look like a huge professional portfolio yet. It is more like a clean starting point that shows what I am practicing.

The main goal was to build the pages with clear HTML structure, readable content, accessible navigation, and one shared CSS file for the main design.
